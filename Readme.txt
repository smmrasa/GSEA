# GSEA
Gene set enrichment analysis 
Works under all version of R (Linux, windows and mac), Tested with R version 4.1.0.
Installation of the packages are automatically done by the script. 

Open R
Set the working directory
Run source ("GeneSetBoxPlot.r") in R terminal
Follow the instruction of the code using the example files.
The expected output is one pdf file and several csv files in a separate folder.  The Run time is fast.
For your data change the input files.
